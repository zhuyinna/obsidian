sqrt函数:
- 思路: 
  - 牛顿迭代法: 
```cpp
int sqrt(int guess){
    if ( guess < 0) {
        cout << "error" << endl;
    } 
    if ( guess == 0 ) {
        cout << 0 << endl;
    }
    double x = guess, eps = 1e-6;
    while ( abs(x * x - guess) > eps ) {
        x = (x + guess / x) / 2;
    }
}
```
