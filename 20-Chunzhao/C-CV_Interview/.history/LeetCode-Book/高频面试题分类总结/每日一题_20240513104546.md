994. 腐烂的橘子

distance[i][j]表示搜索到时所需最短路径(也即时间)
什么情况下跳过, 不执行操作? ( 以下均为或者 )
- 坐标不在合法区域内
- 未访问过: 判断dis[i][j]为-1(`~dis[i][j]`)
- 为空格无橘子: `grid[i][j]==0`

```C++
void bfs(vector<vector<int>>& grid) {
    queue<pair<int, int>> Q;
    // 逐行、逐列遍历grid
        if ( grid[i][j] == 2 ) {

}
```
