#### 68、分别写出 bool， int， float，指针类型的变量a 与“零”的⽐较语句。

```
bool:if ( !a ) or if(a)

int: if ( a == 0) 

float: const EXPRESSION EXP = 0.000001 if ( a <= EXP && a >= -EXP) 

pointer : if ( a != NULL) or if(a == NULL)
```

⽆论是 float 还是 double 类型的变量，都有精度限制。所以⼀定要避免将浮点变量⽤“==”或“！=”与数字⽐较，应该设法转化成“>=”或“<=”形式。
